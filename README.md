## Configuración y Ejecución del Entorno
Para garantizar la correcta ejecución de los scripts prácticos y la generación de gráficas, se recomienda correr el código dentro de un entorno virtual de Python.

### Requisitos Previos
- Python 3.8 o superior instalado en el sistema.

### Pasos de ejecución
1. Clonar el repositorio y ubicarse en el directorio
```shell
git clone https://github.com/alonsogads/Criptografia-y-Seguridad_2026-2
cd Criptografia-y-Seguridad_2026-2/
```

2. Creamos el entorno virtual
```shell
python3 -m venv venv
```

3. Activar el entorno virtual:
- Linux/macOs:
```shell
source venv/bin/activate
```
- Windows (CMD):
```shell
venv\Scripts\activate.bat
```
- Windows (PowerShell):
```shell
.\venv\Scripts\Activate.ps1
```
_El entorno está activo cuando aparezca el prefijo (venv) al inicio de la línea de comandos_

4. Instalamos las dependencias:
```shell
pip install -r requirements.txt
```

5. Ejecutamos los programas del ropositorio.

6. Para salir del entorno virtual, ejecutar el comando `deactivate`, o salir de la interfaz del shell.
